## 概要

---

練習用として架空のコーヒー cafeWeb サイトを制作する。

Web サイトの名称を「THE COFFEE」とする。

Adobe XD を使用してワイヤーフレームとデザインカンプを行い

HTML、SCCS、JavaScript（jQuery） でコーディングを行った、

Web 版とスマホ版（レスポンシブ）に対応

## 使用言語

---

- HTML
- SCSS
- JavaScript (jQuery)
